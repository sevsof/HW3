# HW3
Выполнение домашнего задание 3 по технологиям программирования

Проект состоит из двух Docker-контейнеров
1. Generator-app (генератор данных) - создает CSV-файл с данными о животных с столбцами (возраст, вес, требуется ли животному специальное лечени, вид животного).

2. Reporter-app (аналитик данных) - читает csv и формирует HTML-отчет.

Таблица и html-отчет хранятся в папке `data/`, она создается при запуске контейнера.

## Запуск

Сборка Docker-образов 
./run.sh build_generator   
./run.sh build_reporter    

Генерация таблицы и отчёта
./run.sh run_generator     
./run.sh run_reporter      

Показ структуры 
./run.sh structure

Удаление сгенерированных файлов
./run.sh clear_data

Создание таблицы для локальной отладки 
./run.sh create_local_data

Показ содержимого папки изнутри контейнорв (генератора и аналитика)
./run.sh inside_generator	
./run.sh inside_reporter
